# 🛒 Amazon Clone (HTML & CSS)

## 📌 Project Overview
This project is a static clone of the Amazon homepage built using **HTML and CSS**.  
It replicates the layout, design, and basic structure of the Amazon website to practice frontend development skills.

---

## 🚀 Features
- Responsive navigation bar with logo, search bar, and cart
- Hero section with promotional message
- Product grid layout (shopping section)
- Multi-column footer similar to Amazon
- Clean UI using only HTML & CSS
- Font Awesome icons integration

---

## 🛠️ Technologies Used
- HTML
- CSS
- Font Awesome (for icons)

---

## ▶️ How to Run
1. Clone the repository
2. Run the file:
   Amazon.js
